# Landing Page Generator

可交互 MVP：输入产品名/受众/卖点，生成可编辑落地页模块（Hero、Feature、FAQ、CTA），支持复制 Markdown。

## 功能
- 一键生成 4 大模块
- 模块内联编辑
- Markdown 预览与复制
